MTS Pocket Office - iPhone Private Web App

Features:
- Client PAN, mobile, email, passwords/notes
- Tax follow-up
- Health insurance data
- Vehicle insurance, PUC, fitness expiry
- 30-day due dashboard
- Encrypted local storage using master password
- Encrypted backup/restore

Use:
1. ZIP extract karo.
2. index.html ko iPhone/Mac browser me open karo.
3. iPhone par Safari > Share > Add to Home Screen best rahega.
4. Master password set karo.
5. Regular backup download karte raho.

Important:
- App Store par publish karne ki zarurat nahi.
- Master password bhool gaye to data recover nahi hoga.
